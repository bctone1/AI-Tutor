AI Tutor는 단순한 학습을 넘어서, 다음과 같은 특징을 갖는 '디지털 학습 코치'로 설계됩니다.

개인별 레벨 진단

서비스 최초 이용 시 진단형 테스트를 통해 학생의 전공 이해도 및 수준을 정확히 분석합니다.

맞춤형 학습하기 (기출문제 + AI 생성문제)

교수님이 업로드한 기출문제와 AI가 생성한 새로운 문제를 함께 제공하여 풍부한 학습 기회를 제공합니다.

AI 맞춤 해설

기존 문제은행과 GPT 기반 AI가 생성하는 새로운 해설로 실전 대비력을 강화합니다.

자기주도 학습 관리

풀이 이력, 정답률, 추천 문제 현황을 한눈에 확인할 수 있는 대시보드를 제공합니다.

웹 기반 접근성

학습 장소와 시간의 제약 없이 사용할 수 있는 반응형 웹 인터페이스로 설계됩니다.

빅데이터 피드백

축적된 학습 로그를 분석해 개인 맞춤 피드백을 제공하고 학습 경향을 파악합니다.

사용자별 시스템 및 기능

관리자, 교수자, 학생 각각의 역할에 맞춘 전용 시스템으로 효율적인 학습 생태계를 구축합니다.

관리자 (시스템 운영)

전체 시스템을 관리하고 운영하는 최고 권한자로서, 플랫폼의 안정적인 운영과 성장을 책임집니다.

  • 전체 사용자 계정 및 권한 관리
  • 시스템 성능 모니터링 및 최적화
  • 콘텐츠 품질 관리 및 검증
  • 백업 및 보안 설정 관리
  • 플랫폼 업데이트 및 유지보수

교수자 (해설 품질 관리)

AI가 생성한 해설을 검토하고 개선하여 학습 품질을 높이며, 교육 자료를 관리하는 역할을 담당합니다.

  • 문제 데이터베이스 업로드 및 관리
  • AI 생성 해설의 실시간 검토 및 수정
  • 문제별 맞춤형 해설 직접 작성
  • 교육 자료 및 참고서 RAG 시스템 업로드
  • 해설 버전 관리 및 품질 보증
  • 학생별 학습 현황 모니터링

학생 (맞춤형 학습)

개인화된 학습 환경에서 AI 튜터와 함께 효율적으로 학습하며 실력을 향상시킵니다.

  • 개인 수준 진단 테스트 참여
  • AI 튜터와 실시간 대화형 학습
  • 맞춤형 문제 추천 및 풀이
  • 상세한 해설 및 피드백 확인
  • 학습 진도 및 성취도 추적
  • 취약점 분석 및 집중 학습 계획

핵심 기술 및 시스템 구조

데이터, AI 기술, 사용자 인터페이스가 유기적으로 결합된 4계층 맞춤형 학습 플랫폼입니다.

하이브리드 데이터 시스템
(RAG + RDBMS)

RAG와 관계형 데이터베이스를 결합하여 효율적인 문제 관리와 정확한 정답 제공을 동시에 구현합니다.

  • Vector DB를 통한 고속 문제 검색
  • 관계형 DB로 정답 데이터 정확성 보장
  • 기출문제 데이터 표준화 및 구조화
  • 교육 자료 품질 검증 시스템

LLM 연동 AI 엔진
(ChatGPT API)

ChatGPT API와 RAG 기술을 활용하여 개인별 수준에 맞춘 맞춤형 해설을 실시간으로 생성합니다.

  • OpenAI ChatGPT API 통합 활용
  • 벡터 데이터베이스 기반 유사도 검색
  • 컨텍스트 기반 개인화 해설 생성
  • 높은 유사도 순서 기반 문제 추천

AI 문제 생성 엔진
(GPT 파인튜닝)

기존 기출문제를 학습한 AI 모델이 학생 수준에 맞는 새로운 문제를 무한 생성하여 학습 다양성을 제공합니다.

  • 기존 기출문제 데이터셋 학습
  • 파라미터 조정을 통한 난이도별 문제 생성
  • 프롬프트 엔지니어링으로 문제 품질 최적화
  • 실시간 문제 생성 및 검증 시스템

4계층 시스템
아키텍처

확장 가능한 마이크로서비스 구조로 설계되어 각 기능이 독립적으로 동작하며, 시스템 확장성과 안정성을 보장합니다.

  • 데이터 저장/관리 계층 (Vector DB + RDBMS)
  • AI 처리 계층 (RAG + LLM + 문제생성)
  • 비즈니스 로직 계층 (학습 관리 시스템)
  • 사용자 인터페이스 계층 (웹 기반 UI)

1

학과별 기출문제 수집 및 전처리

각 학과의 기출문제를 수집하고 AI 학습에 적합한 형태로 정제하여 고품질 데이터셋을 구축합니다.

  • 학과별 기출문제 분류 및 태깅
  • 문제 유형별 구조 분석
  • 데이터 품질 검증 및 정제
2

기출문제 패턴 분석 및 AI 학습

수집된 학과별 기출문제를 분석하여 문제 패턴을 파악하고, AI 모델이 이를 학습하도록 훈련시킵니다.

  • 학과별 문제 패턴 및 구조 분석
  • 출제 경향 및 문제 스타일 학습
  • 전문 분야 지식 모델링
3

프롬프트 및 파라미터 최적화

기존 기출문제에 맞는 새로운 문제를 생성하기 위해 프롬프트와 생성 파라미터를 세밀하게 조정합니다.

  • 학과별 맞춤 프롬프트 템플릿 개발
  • 문제 생성 파라미터 조정
  • 기출문제 스타일 일치성 최적화

RAG 기반 해설 품질 향상 프로세스

교육 자료를 RAG 시스템에 업로드하여 LLM이 정확하고 품질 높은 해설을 생성하는 5단계 시스템입니다.
1

교육 자료 업로드

표준 교재, 참고서, 강의 노트 등 신뢰할 수 있는 교육 자료를 RAG 데이터베이스에 업로드합니다.

  • 텍스트 파일, PDF, Word 문서 지원
  • 다양한 형태의 교육 자료 처리
  • 자동 문서 분석 및 분류
2

벡터 데이터베이스 구축

업로드된 교육 자료를 벡터화하여 RAG 시스템에 저장하고 의미론적 검색이 가능하도록 최적화합니다.

  • 고차원 벡터 임베딩 생성
  • 의미론적 유사도 기반 인덱싱
  • 실시간 검색 최적화
3

컨텍스트 기반 해설 생성

문제가 주어지면 RAG 시스템이 관련 교육 자료를 검색하고, 이를 컨텍스트로 활용하여 정확한 해설을 생성합니다.

AI 튜터

정답입니다! ③ 척수신경은 중추신경계에 포함되지 않습니다.

해설: 중추신경계(CNS)는 뇌와 척수로 구성됩니다. 대뇌, 소뇌, 뇌간은 모두 뇌의 일부이며, 척수와 함께 중추신경계를 형성합니다. 반면, 척수신경은 말초신경계(PNS)에 속합니다.
